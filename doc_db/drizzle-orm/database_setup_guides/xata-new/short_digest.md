## Get Started with Drizzle and Xata

1. Install postgres package
2. Set `DATABASE_URL` environment variable with Xata connection string
3. Connect Drizzle using postgres driver
4. Define tables in schema
5. Create `drizzle.config.ts` with `dialect: 'postgresql'` and `DATABASE_URL`
6. Run migrations
7. Seed and query database with TypeScript
8. Execute with tsx
