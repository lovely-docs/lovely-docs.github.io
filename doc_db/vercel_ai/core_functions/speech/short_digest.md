The `generateSpeech` function converts text to audio. Accepts `model`, `text`, optional `voice`, `language`, `providerOptions`, `abortSignal`, and `headers`. Returns audio data and warnings. Throws `AI_NoSpeechGeneratedError` on failure. Supports OpenAI (tts-1, tts-1-hd, gpt-4o-mini-tts), ElevenLabs (eleven_v3, eleven_multilingual_v2, eleven_flash_v2_5, eleven_flash_v2, eleven_turbo_v2_5, eleven_turbo_v2), LMNT (aurora, blizzard), and Hume (default) models.
