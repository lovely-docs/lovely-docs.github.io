SvelteKit emits OpenTelemetry spans for `handle`, `load`, form actions, and remote functions (experimental, opt-in via config). Set up tracing in `src/instrumentation.server.ts`. Access spans via `event.tracing.root` and `event.tracing.current` to add custom attributes. Quickstart with Jaeger using `@opentelemetry/sdk-node` and `@opentelemetry/exporter-trace-otlp-proto`.
