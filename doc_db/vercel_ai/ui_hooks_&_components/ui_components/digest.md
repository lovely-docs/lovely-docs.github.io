AI SDK UI provides a collection of components and patterns for building AI-powered user interfaces. The documentation covers:

**Core Components:**
- Overview: Introduction to AI SDK UI capabilities
- Chatbot: Building conversational interfaces with message handling
- Chatbot Message Persistence: Storing and loading chat history
- Chatbot Tool Usage: Integrating tool calling capabilities into chatbots
- Completion: Text completion interfaces
- Object Generation: Generating structured objects through UI
- Streaming Data: Streaming data from backend to frontend

**Advanced Topics:**
- Reading UI Message Streams: Consuming message streams for terminal UIs, custom clients, and server components
- Error Handling: Managing errors in AI-powered interfaces
- Stream Protocol: The underlying protocol defining data transmission from backend to AI SDK UI frontend

The documentation provides practical guides for integrating each component type into applications.
