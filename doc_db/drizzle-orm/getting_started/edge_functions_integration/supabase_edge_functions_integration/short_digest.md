Create schema in `src/schema.ts`, configure `drizzle.config.ts`, initialize Supabase with `supabase init`, generate migrations with `drizzle-kit generate`, apply with `supabase migration up`. Create Edge Function with `supabase functions new`, add imports to `deno.json`, connect database using `postgres` client and `drizzle()` wrapper with `SUPABASE_DB_URL` env var. Test locally with `supabase functions serve --no-verify-jwt`, then link to hosted project with `supabase link`, push schema with `supabase db push`, set `DATABASE_URL` secret, and deploy with `supabase functions deploy`.
