**DOM & State Utilities**: Reactive wrappers for browser APIs and state management. Focus/visibility tracking (`activeElement`, `IsDocumentVisible`, `IsFocusWithin`). Element measurements (`ElementRect`, `ElementSize`, `ScrollState`). Animation/timing (`AnimationFrames`, `useInterval`). User interaction (`PressedKeys`, `IsIdle`, `onClickOutside`). Async data (`resource` with cancellation/debounce, `PersistedState` with localStorage sync). State management (`Context`, `Debounced`, `Throttled`, `Previous`, `StateHistory`, `IsMounted`). Reactive utilities (`watch`, `extract`, `boolAttr`). DOM observers (`useIntersectionObserver`, `useResizeObserver`, `useMutationObserver`, `IsInViewport`). Event handling (`useEventListener`, `onCleanup`). Specialized (`TextareaAutosize`, `FiniteStateMachine`, `useGeolocation`, `useSearchParams` with schema validation). Install: `npm install runed`. Use with `$state`/`$effect` in Svelte components.
