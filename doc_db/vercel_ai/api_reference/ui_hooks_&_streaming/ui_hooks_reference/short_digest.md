AI SDK UI provides hooks and utilities for building chat/completion/assistant apps across React, Svelte, and Vue.js. Core hooks: `useChat`, `useCompletion`, `useObject`. Utilities: `convertToModelMessages`, `pruneMessages`, `createUIMessageStream`, `createUIMessageStreamResponse`, `pipeUIMessageStreamToResponse`, `readUIMessageStream`. Framework support varies: React has all features, Svelte has all, Vue.js missing `useObject`.
