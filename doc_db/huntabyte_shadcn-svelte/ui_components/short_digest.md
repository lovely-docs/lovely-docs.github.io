## UI Components

60+ composable, accessible components built on Bits UI, Embla Carousel, Vaul, and other primitives.

**Installation**: `npx shadcn-svelte@latest add <component> -y -o`

**Categories**:
- **Layout**: Sidebar, Card, Resizable, Scroll Area, Separator, Empty
- **Navigation**: Breadcrumb, Navigation Menu, Pagination, Tabs, Menubar
- **Forms**: Input, Textarea, Checkbox, Radio, Switch, Select, Slider, Toggle, Field
- **Dialogs**: Dialog, Alert Dialog, Drawer, Popover, Hover Card, Sheet, Tooltip
- **Menus**: Dropdown Menu, Context Menu, Combobox
- **Data**: Table, Data Table, Carousel, Chart, Badge, Avatar, Item
- **Feedback**: Alert, Progress, Spinner, Sonner (toasts)
- **Utilities**: Accordion, Aspect Ratio, Button, Collapsible, Calendar, Date Picker, Kbd, Skeleton

**Form Integration**: sveltekit-superforms + Zod for type-safe validation with error handling.

**Composition**: Sub-components pattern (e.g., `Card.Root`, `Card.Header`, `Card.Content`).

**Accessibility**: WAI-ARIA compliant, keyboard navigation, semantic HTML.

**Theming**: Tailwind CSS utilities, CSS variables, dark mode support.
