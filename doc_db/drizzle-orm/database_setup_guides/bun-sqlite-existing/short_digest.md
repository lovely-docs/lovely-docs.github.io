Install `drizzle-orm`, `drizzle-kit`, `tsx`, and `@types/bun`. Set `DB_FILE_NAME` env variable pointing to SQLite file. Create `drizzle.config.ts` with SQLite dialect. Introspect existing database, transfer schema to file, connect via bun:sqlite, query database, run with `bun src/index.ts`. Optionally update schema and apply migrations.
