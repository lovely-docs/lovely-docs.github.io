Registry items define components, styles, themes via JSON schema with `name`, `type`, `files`, `dependencies`, `registryDependencies`, `cssVars`, and `css`. Root `registry.json` configures the registry with `items`, `aliases` (import path mapping), and `overrideDependencies`. Build with `shadcn-svelte registry build`, serve locally or deploy publicly, install with `npx shadcn-svelte@latest add <url> -y -o`.
