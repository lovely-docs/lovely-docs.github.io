Combines two Time Field components for start/end time input. Supports two-way and fully controlled binding for `value` (TimeRange with start/end TimeValue) and `placeholder`. Configurable via `minValue`, `maxValue`, `granularity` ('hour'|'minute'|'second'), `hourCycle` ('12'|'24'), `locale`, `disabled`, `readonly`, `required`, `readonlySegments`, validation, and callbacks. Components: Root (main), Input (type: 'start'|'end'), Segment (part: hour|minute|second|dayPeriod|timeZoneName|literal), Label.
