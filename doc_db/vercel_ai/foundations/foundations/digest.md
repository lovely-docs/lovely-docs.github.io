This is a foundational documentation section covering core concepts essential to working with the AI SDK. It provides an organized entry point to six key topic areas:

**Overview** - Introduces foundational concepts around AI and LLMs, establishing baseline knowledge needed for the SDK.

**Providers and Models** - Documents the available AI providers and models that can be integrated with the SDK, helping developers choose appropriate services for their use cases.

**Prompts** - Explains how prompts are structured, used, and defined within the SDK framework.

**Tools** - Covers the tools system in the SDK, enabling developers to extend AI capabilities with custom functions and integrations.

**Streaming** - Details why streaming is important for AI applications, covering performance and user experience benefits.

**Agents** - Provides guidance on building autonomous agents using the SDK, including agent architecture and implementation patterns.

Each topic is presented as a separate documentation page accessible through the index cards interface.
