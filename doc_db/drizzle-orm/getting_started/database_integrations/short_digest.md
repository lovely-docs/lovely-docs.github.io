Setup guides for Neon (neon-http), Supabase (postgres-js), Turso (libSQL/SQLite), Vercel Postgres, Xata (postgres-js), and Nile (multi-tenant). Each includes: driver installation, connection config, schema definition with pgTable/sqliteTable, drizzle.config.ts setup, migration commands, and CRUD examples (insert/select/update/delete with joins, filtering, pagination). Nile uses AsyncLocalStorage + tenantDB() wrapper for automatic per-tenant isolation via nile.tenant_id context variable.
