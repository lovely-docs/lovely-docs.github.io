## Tooltip

Displays information on hover or focus.

```svelte
<Tooltip.Provider>
  <Tooltip.Root>
    <Tooltip.Trigger>Hover</Tooltip.Trigger>
    <Tooltip.Content>
      <p>Add to library</p>
    </Tooltip.Content>
  </Tooltip.Root>
</Tooltip.Provider>
```

Install: `pnpm dlx shadcn-svelte@latest add tooltip`
