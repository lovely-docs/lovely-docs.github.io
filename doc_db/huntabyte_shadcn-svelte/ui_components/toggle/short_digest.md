## Toggle

Two-state button component.

**Install:** `npx shadcn-svelte@latest add toggle -y -o`

**Basic usage:**
```svelte
<Toggle>Toggle</Toggle>
```

**Props:** `variant` ("default" | "outline"), `size` ("sm" | "default" | "lg"), `disabled`, `aria-label`

**Examples:** default with icon, outline variant, with text, small/large sizes, disabled state
