**60+ Svelte UI components** (Accordion, Button, Form controls, Card, Sidebar, Navigation, Overlays, DataTable, Calendar, Chart, etc.) installed via `npx shadcn-svelte@latest add <component> -y -o`. Composable subcomponents, Svelte 5 reactivity, Tailwind CSS, ARIA accessible, dark mode via CSS variables (OKLCH format). Setup: init project with `npx shadcn-svelte@latest init`, configure components.json (baseColor, aliases, registry), add Tailwind CSS. Dark mode: mode-watcher package with toggleMode/setMode/resetMode. Custom registries: JSON schema with registry items, build with CLI, serve and install. Migrations: Svelte 4→5 (update deps, simplify utils, migrate components), Tailwind v3→v4 (Vite plugin, tw-animate-css, @theme inline). IDE extensions for VSCode/JetBrains. JavaScript support via components.json flag.
