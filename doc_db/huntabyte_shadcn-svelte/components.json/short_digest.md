Configuration file for shadcn-svelte CLI (optional, only needed for CLI-based component installation).

Create with `npx shadcn-svelte@latest init`

Key settings:
- `$schema`: JSON schema reference
- `tailwind.css`: Path to Tailwind import file
- `tailwind.baseColor`: Default palette (gray|neutral|slate|stone|zinc)
- `aliases`: Path aliases for lib, utils, components, ui, hooks
- `typescript`: Enable/disable or specify custom config path
- `registry`: Component registry URL
