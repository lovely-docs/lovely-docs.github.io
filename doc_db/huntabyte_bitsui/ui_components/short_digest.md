## UI Components

Accessible, composable Svelte components with state management, keyboard navigation, and Floating UI positioning.

**Categories**: Form controls (Checkbox, Select, Slider, PIN Input, Rating), Date/Time (Calendar, DateField, DatePicker), Disclosure (Accordion, Collapsible, Tabs), Menus (Dropdown, Context Menu, Command), Dialogs (Dialog, Popover, Tooltip), Progress (Progress, Meter, Pagination), Layout (Separator, ScrollArea, Avatar, Button).

**Key Features**: Two-way binding + function bindings, compound sub-components, data attributes + CSS variables, WAI-ARIA accessibility, Floating UI positioning, Svelte transitions via `forceMount`, form integration with hidden inputs, validation (min/max/custom), keyboard navigation.

**Common Patterns**: Reusable wrapper components, scroll management with buttons/native scrollbars, nested structures with depth tracking, search/filtering with custom algorithms, multi-selection via arrays.
