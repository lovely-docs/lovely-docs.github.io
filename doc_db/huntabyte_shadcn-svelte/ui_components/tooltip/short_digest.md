## Tooltip

Popup displaying information on hover or focus.

```svelte
<Tooltip.Provider>
  <Tooltip.Root>
    <Tooltip.Trigger>Hover</Tooltip.Trigger>
    <Tooltip.Content>Add to library</Tooltip.Content>
  </Tooltip.Root>
</Tooltip.Provider>
```

Install: `npx shadcn-svelte@latest add tooltip -y -o`
