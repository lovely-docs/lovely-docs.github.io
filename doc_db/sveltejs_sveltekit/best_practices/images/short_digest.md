Three image optimization approaches: Vite's built-in asset handling with hashing, `@sveltejs/enhanced-img` plugin for automatic format conversion and responsive sizing, and CDN-based dynamic loading for runtime images. Setup enhanced-img with `npm i -D @sveltejs/enhanced-img` and add to vite config. Use `<enhanced:img src="./image.jpg" alt="text" />` for static images or `import img from './image.jpg?enhanced'` for dynamic imports. Specify `sizes` for responsive images and `?w=1280;640;400` for custom widths. Best practices: provide 2x resolution originals, use `fetchpriority="high"` for LCP images, set `width`/`height` to prevent layout shift, avoid `em`/`rem` in sizes.
