Install `@sveltejs/adapter-node` and add to `svelte.config.js`. Build with `npm run build`, run with `node build`. Configure via env vars: `PORT`/`HOST`/`SOCKET_PATH` for connections, `ORIGIN`/`PROTOCOL_HEADER`/`HOST_HEADER` for URL detection, `ADDRESS_HEADER`/`XFF_DEPTH` for client IP behind proxies, `BODY_SIZE_LIMIT` for request size, `SHUTDOWN_TIMEOUT` for graceful shutdown. Adapter options: `out` (build dir), `precompress` (gzip/brotli), `envPrefix` (env var prefix). Graceful shutdown on SIGTERM/SIGINT with `sveltekit:shutdown` event. Supports systemd socket activation. Export `handler.js` for custom servers (Express, Polka, etc.).
