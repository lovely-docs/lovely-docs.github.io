Four core functions: `generateText()` (non-interactive), `streamText()` (interactive streaming), `generateObject()` (structured data), `streamObject()` (streaming objects). Tool calling with multi-step support, dynamic tools, error handling, repair. MCP server integration. Reranking, image generation, transcription, speech generation. Middleware for intercepting model calls. Provider management via custom providers and registries. Error handling patterns. Mock providers for testing. OpenTelemetry telemetry support.
