## Integer Types
- **integer** (int, int4): 4-byte signed integer
- **smallint** (int2): 2-byte signed integer  
- **bigint** (int8): 8-byte signed integer, use `mode: 'number'` for 2^31-2^53 range
- **serial** (serial4): Auto-incrementing 4-byte integer
- **smallserial** (serial2): Auto-incrementing 2-byte integer
- **bigserial** (serial8): Auto-incrementing 8-byte integer

```ts
import { integer, smallint, bigint, serial, pgTable } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  int: integer().default(10),
  smallint: smallint(),
  bigint: bigint({ mode: 'number' }),
  serial: serial()
});
```

## Text Types
- **text**: Unlimited variable-length string, supports `enum` config for type inference
- **varchar(n)**: Variable-length up to n chars, length optional, supports enum
- **char(n)**: Fixed-length blank-padded up to n chars, length optional, supports enum

```ts
import { text, varchar, char, pgTable } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  text: text({ enum: ["a", "b"] }),
  varchar: varchar({ length: 256 }),
  char: char({ length: 10 })
});
```

## Numeric Types
- **numeric/decimal**: Exact precision, up to 131072 digits before decimal, 16383 after. Supports `mode: 'number' | 'bigint'` and `precision`, `scale` options
- **real** (float4): 4-byte floating-point
- **doublePrecision** (float8): 8-byte floating-point

```ts
import { numeric, real, doublePrecision, pgTable } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  numeric: numeric({ precision: 100, scale: 20 }),
  real: real().default(10.10),
  double: doublePrecision().default(10.10)
});
```

## JSON Types
- **json**: Textual JSON (RFC 7159)
- **jsonb**: Binary JSON (decomposed)
- Both support `.$type<T>()` for compile-time type inference

```ts
import { json, jsonb, pgTable } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  json: json().$type<{ foo: string }>().default({ foo: "bar" }),
  jsonb: jsonb().$type<string[]>()
});
```

## Date/Time Types
- **time**: Time of day, options: `withTimezone`, `precision`
- **timestamp**: Date and time, options: `precision`, `withTimezone`, `mode: 'date' | 'string'` (default 'date')
- **date**: Calendar date, options: `mode: 'date' | 'string'`
- **interval**: Time span, options: `fields`, `precision`

```ts
import { time, timestamp, date, interval, pgTable } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  time: time({ withTimezone: true, precision: 6 }),
  timestamp: timestamp({ mode: "date" }).defaultNow(),
  date: date({ mode: "string" }),
  interval: interval({ fields: 'day', precision: 6 })
});
```

## Geometric Types
- **point**: Two modes: `tuple` (default [x,y]) or `xy` ({x,y})
- **line**: Two modes: `tuple` (default [a,b,c]) or `abc` ({a,b,c})

```ts
import { point, line, pgTable } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  point: point({ mode: 'xy' }),
  line: line({ mode: 'abc' })
});
```

## Enum Type
```ts
import { pgEnum, pgTable } from "drizzle-orm/pg-core";
export const moodEnum = pgEnum('mood', ['sad', 'ok', 'happy']);
export const table = pgTable('table', { mood: moodEnum() });
```

## Identity Columns (drizzle-orm@0.32.0+, drizzle-kit@0.23.0+)
- **GENERATED ALWAYS AS IDENTITY**: Database always generates, manual insertion requires OVERRIDING SYSTEM VALUE
- **GENERATED BY DEFAULT AS IDENTITY**: Database generates by default, manual values allowed

```ts
import { pgTable, integer, text } from 'drizzle-orm/pg-core';
export const table = pgTable("table", {
  id: integer().primaryKey().generatedAlwaysAsIdentity({ startWith: 1000 }),
  name: text().notNull()
});
```

## Default Values & Runtime Defaults
- **`.default(value)`**: Static default value
- **`.$defaultFn(fn)`**: Runtime default generation (uuid, cuid, etc), insert only
- **`.$onUpdateFn(fn)`**: Runtime value on update, also called on insert if no default

```ts
import { integer, timestamp, text, pgTable, uuid } from "drizzle-orm/pg-core";
export const table = pgTable('table', {
  id: uuid().defaultRandom(),
  counter: integer().default(1).$onUpdateFn(() => sql`counter + 1`),
  updatedAt: timestamp().$onUpdate(() => new Date()),
  customId: text().$defaultFn(() => createId())
});
```

## Column Customization
- **`.$type<T>()`**: Customize inferred type for branded/unknown types
- **`.notNull()`**: NOT NULL constraint
- **`.primaryKey()`**: PRIMARY KEY constraint

```ts
type UserId = number & { __brand: 'user_id' };
const users = pgTable('users', {
  id: serial().$type<UserId>().primaryKey(),
  name: text().notNull()
});
```

## Notes
- Column names generated from TypeScript keys; use database aliases or `casing` parameter for custom mapping
- Custom types available via custom type creation
- `timestamp with timezone` stored in UTC, converted based on Postgres instance timezone
