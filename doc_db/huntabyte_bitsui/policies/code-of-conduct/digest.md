## Code of Conduct

Community expectations for respectful and productive participation in Bits UI.

**Core Principle:** Treat others with respect and contribute positively.

### Community Guidelines

- **Be Respectful** – Treat everyone with respect, even in disagreement. No personal attacks, harassment, or aggression.
- **Be Constructive** – Focus on productive discussions and feedback. Avoid negativity without solutions.
- **Follow Contribution Guidelines** – Feature requests must start as discussions before implementation. Issues should be limited to confirmed bugs or accepted feature requests ready for development.
- **Keep Discussions On-Topic** – Use appropriate channels for discussions, issues, and pull requests to maintain organization and efficiency.

### Unacceptable Behavior

- Personal attacks, insults, or threats
- Harassment, bullying, or intimidation
- Discrimination based on any personal attribute
- Spamming, trolling, or deliberately disruptive behavior
- Sharing private information without consent

### Enforcement

Maintainers have the right to remove, edit, or reject comments, commits, code, issues, and contributions that violate this Code of Conduct. Serious violations may result in temporary or permanent bans.

Report unacceptable behavior to project maintainers.

**Scope:** Applies to all project spaces including GitHub (issues, discussions, pull requests), Discord chat channels, and official communication platforms.
