Creates MCP client for connecting to MCP servers with tools, resources, prompts, and elicitation support. Configure with transport (stdio/SSE/HTTP), optional OAuth, and error handlers. Methods: `tools()`, `listResources()`, `readResource(uri)`, `listResourceTemplates()`, `listPrompts()`, `getPrompt(name)`, `onElicitationRequest(schema, handler)`, `close()`. Throws `MCPClientError` for connection/protocol issues.
