## Typography Utility Classes

No default styles shipped. Use Tailwind utilities for text styling:

**Headings**: h1 (text-4xl lg:text-5xl), h2 (text-3xl with border-b), h3 (text-2xl), h4 (text-xl)

**Text**: p (leading-7 with margin), Lead (text-xl muted), Large (text-lg semibold), Small (text-sm), Muted (text-sm muted-foreground)

**Elements**: blockquote (border-l-2 italic), inline code (bg-muted rounded), ul (list-disc), table (w-full with borders and alternating row colors)

All use Tailwind classes like `scroll-m-20`, `tracking-tight`, `font-semibold`, etc.
