Carousel component built on Embla with motion and swipe. Install with `npx shadcn-svelte@latest add carousel -y -o`. Use `<Carousel.Root>`, `<Carousel.Content>`, `<Carousel.Item>`, `<Carousel.Previous />`, `<Carousel.Next />`. Control sizing with `basis` classes, spacing with `pl-` and `-ml-` utilities, orientation with `orientation="vertical|horizontal"`. Pass options via `opts` prop. Access API with `setApi` callback to track current slide and listen to events. Add plugins like Autoplay via `plugins` prop.
