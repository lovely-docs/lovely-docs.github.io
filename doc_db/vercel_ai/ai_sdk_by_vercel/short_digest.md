TypeScript toolkit for building AI applications with LLMs across React, Next.js, Vue, Svelte, Node.js. Standardizes integration across multiple model providers. Two main libraries: AI SDK Core (text generation, structured objects, tool calls, agents) and AI SDK UI (framework-agnostic hooks for chat/generative UI). Templates available for various use cases. Full documentation accessible at ai-sdk.dev/llms.txt for LLM-based queries.
