**Reactive variables**: top-level vars auto-reactive; mutations need reassignment. **Reactive statements**: `$:` prefix, compile-time dependency detection, topological ordering. **Props**: `export let`, `$$props`/`$$restProps`. **Events**: `on:` directive with modifiers, `createEventDispatcher` for custom events. **Slots**: default/named with fallback, `let:` for data passing, `$$slots` to check presence. **Dynamic components**: `<svelte:component this={ref} />`. **Imperative API**: constructor with target/props/context, `$set()`, `$on()`, `$destroy()` methods.
