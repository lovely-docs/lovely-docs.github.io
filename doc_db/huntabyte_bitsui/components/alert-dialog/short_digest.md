Modal dialog component with compound structure (Root, Trigger, Portal, Overlay, Content, Title, Description, Cancel, Action). Supports controlled/uncontrolled state, focus trapping, scroll locking, escape key handling, outside interaction customization, and nested dialogs. Key props: `open` (bindable), `trapFocus`, `preventScroll`, `escapeKeydownBehavior`, `interactOutsideBehavior`, `onOpenAutoFocus`, `onCloseAutoFocus`. Data attributes and CSS variables for styling nested dialogs.
