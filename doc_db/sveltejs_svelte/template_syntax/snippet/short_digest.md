# Snippets

Reusable markup chunks declared with `{#snippet name(params)}...{/snippet}` and rendered via `{@render name(args)}`. Can reference enclosing scope, support recursion, and be passed as component props (explicit or implicit). Optional snippets use `?.()` or `#if` checks. Type with `Snippet<[ParamTypes]>` from `'svelte'`. Implicit `children` snippet captures non-snippet content in components. Can be exported from `<script module>` (Svelte 5.5+). Replaces deprecated slots.
