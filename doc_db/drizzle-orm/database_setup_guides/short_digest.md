Setup guides for 30+ databases: PostgreSQL, MySQL, SQLite, Neon, Supabase, PlanetScale, Turso, Cloudflare D1, Bun, Expo, and more. Standard pattern: install driver → set DATABASE_URL → create drizzle.config.ts → define schema → generate migrations → initialize client → query. Includes introspection for existing databases, multi-tenant patterns (Nile), React Native support (Expo/OP-SQLite), and edge computing (Cloudflare Workers).
