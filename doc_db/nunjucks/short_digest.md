**Nunjucks**: JavaScript templating engine (Node/browser), Jinja2-inspired. Configure with `nunjucks.configure(path, opts)`, render with `render()`/`renderString()`/`compile()`. Environment manages loaders (FileSystem, Web, custom), filters, extensions, globals. Template syntax: variables `{{ foo }}`, filters `{{ foo | upper }}`, inheritance `{% extends %}` + `{% block %}`, control flow `{% if %}`/`{% for %}`, composition `{% include %}`/`{% import %}`/`{% macro %}`, async `{% asyncEach %}`/`{% asyncAll %}`. 40+ builtin filters.
